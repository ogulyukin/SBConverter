Внимание! Поскольку Сбербанк принимает файлы только в ANSI кодировке, 
все исходные файлы должны быть тоже сохранены в ANSI кодировке.

Файл с кодами лицевых счетов должен быть скачен с сайта ГИС ЖКХ
пример: "Экспорт ЕЛС от 25.08.2021 12-24.xlsx"

Файл с начислениями в старом формате должен иметь расширение .txt

Код фиас установлен по умолчанию на код г. Черняховск Калининградской обл. Если вы находитесь
в другом населенном пункте - узнайте код своего населенного пункта и встатвьте его вместо 
установленного по умолчанию.

Программа запомнит все введенные вами ранее данные.
